# Stock Price Tracker

## Description
A Stock Price Tracker is a tool to fetch and display stock prices, historical data, and perform basic analysis. This task provides a simple command-line interface for tracking stock information.

## Features
- Fetch stock prices by ticker symbol
- Display latest price and company details
- Show historical price data and trends
- Basic performance metrics and summary

## Technologies Used
- Python 3
- Standard library only

## Files
- `stock.py` — main stock tracker script

## How to Run

1. Open terminal in the `Task-2-Stock` folder
2. Run:

```bash
python stock.py
```

3. Enter a stock ticker symbol when prompted.

## Notes
- Internet access is required for live stock data retrieval.
- The script uses fallback sample data if live data cannot be fetched.
