# Stock Price Tracker

## Description
A Stock Price Tracker is a tool to fetch and display stock prices, historical data, and perform basic analysis. This task provides a simple command-line interface for tracking stock information.

## Features
- Fetch stock prices by ticker symbol
- Display latest price and company details
- Show historical price data and trends
- Basic performance metrics and summary

## Technologies Used
- Python
- Requests (for API or web requests)
- Pandas (optional for data display)

## How to Run

1. Clone the repository
2. Open terminal in the `Task-2-Stock` folder
3. Install dependencies if required:

```bash
pip install -r requirements.txt
```

4. Run the stock tracker script:

```bash
python stock_tracker.py
```

5. Enter a stock ticker symbol when prompted.

## Notes
- Update API keys or data source settings in the script as needed.
- Make sure internet access is available for live stock data.
