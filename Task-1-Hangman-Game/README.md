# Hangman Game

## Description
Hangman is a simple word guessing game developed using Python. The player has to guess the hidden word one letter at a time before running out of attempts.

## Features
- Random word selection
- Letter-by-letter guessing
- Tracks incorrect guesses
- Win and lose conditions
- Console-based gameplay

## Technologies Used
- Python

## How to Run

1. Clone the repository
2. Open terminal in the project folder
3. Run the command:

```bash
python hangman.py
